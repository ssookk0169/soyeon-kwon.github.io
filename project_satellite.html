위성 영상 기반 벼 재배 면적 탐지 AI 프로젝트

2024.12.25 - 2025.07.25

프로젝트 개요

기존의 수작업 기반 재배 면적 조사는 막대한 비용과 시간이 소요됩니다. 이를 해결하기 위해 초고해상도 위성 영상을 활용하여 벼 재배지를 자동으로 탐지하는 딥러닝 모델을 구축하였습니다.

수행 내용

  • 초고해상도 위성 데이터 전처리 및 데이터 증강(Augmentation) 수행
  • U-Net 아키텍처 기반 세그멘테이션 모델 설계
  • 정규화 식생 지수(NDVI) 정보를 활용한 탐지 필터링 로직 개발
